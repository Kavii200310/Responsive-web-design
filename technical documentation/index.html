Getting Started With Java

Steps to get started:

  1. Install the JDK appropriate for your operating system.
  2. Use an IDE such as IntelliJ IDEA, Eclipse, or NetBeans. You can also use a text editor like VS Code.
  3. Configure your environment variables (e.g., `JAVA_HOME`) and ensure `javac` and `java` commands are accessible from the terminal.
  4. Create your first Java file using the `.java` extension, such as `HelloWorld.java`.
Java Entry Point

The entry point of a Java application is the `main` method.

This method must have the following signature:

  • public: Makes the method accessible to the JVM.
  • static: Allows the method to run without creating an instance of the class.
  • void: Indicates the method does not return any value.
  • String[] args: Allows passing command-line arguments to the program.
                
    public class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Welcome to Java!");
        }
    }
Printing to the Console

Java provides the `System.out.println` method to print text to the console.

Printing to the console is primarily used for debugging, displaying program results, and providing information to the user. However, it is not recommended for production-level logging, where robust logging frameworks are preferred.

  1. Print a simple message:
    2. System.out.println("Hello, Java!");
  2. Print variables:
    3. 
    int age = 25;
    System.out.println("Age: " + age);
  3. Print without a newline:
    4. System.out.print("This is inline text.");
Declaring Functions

In Java, functions (or methods) are blocks of code that perform specific tasks.

Functions must be declared within a class.

General syntax:

                
    accessModifier returnType methodName(parameters) {
        // Method body
    }
Example:
                
    public class Calculator {
        public int add(int a, int b) {
            return a + b;
        }
    
        public static void main(String[] args) {
            Calculator calc = new Calculator();
            int result = calc.add(5, 7);
            System.out.println("Sum: " + result);
        }
    }
Object-Oriented Programming

Java is a fully object-oriented programming language, emphasizing the use of objects and classes to structure programs.

Key principles of OOP in Java include:

  1. Encapsulation: Wrapping data and methods together in a class to protect and manage them.
  2. Inheritance: Creating new classes from existing ones to promote reusability.
  3. Polymorphism: Using a single interface to represent different types of behavior.
  4. Abstraction: Hiding implementation details while exposing functionality.
